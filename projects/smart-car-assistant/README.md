# 🚗 Smart Car Buying Assistant

![AI](https://img.shields.io/badge/Project-AI%20Agent-blue)

---

## ❗ Problem

Buying a car online requires searching multiple websites, comparing listings, and guessing whether a deal is good.

---

## ✅ Solution

This AI system:

* Collects listings
* Cleans and standardizes data
* Uses NLP to understand descriptions
* Uses RNN to analyze price trends
* Ranks best deals
