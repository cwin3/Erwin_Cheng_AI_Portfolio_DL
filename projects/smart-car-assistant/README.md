# 🚗 Smart Car Buying Assistant

![AI](https://img.shields.io/badge/Project-AI%20Agent-blue)

---

## ❗ Problem

Buying a car online is fragmented and time-consuming. Users must:
- Search multiple sites
- Compare inconsistent data
- Risk overpaying

---

## ✅ Solution

This AI system:

* Collects listings
* Cleans and standardizes data
* Uses NLP to understand descriptions
* Uses RNN to analyze price trends
* Ranks best deals
